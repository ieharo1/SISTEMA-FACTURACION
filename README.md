# DeberJFrame

Proyecto Java para prácticas de programación orientada a objetos con interfaces gráficas.

## Descripción

Aplicación Java que utiliza JFrame para crear interfaces gráficas y practicar conceptos de POO.

## Características

- Uso de JFrame para ventanas
- Implementación de eventos
- Práctica de programación orientada a objetos

## Stack Tecnológico

- Java SE
- Swing (JFrame)
